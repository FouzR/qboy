QBoy
====

A Gameboy Emulator in C++ with Qt

Dependencies
------------
- C++
- Qt 4 or 5

Features
--------

- Emulates the monochrome gameboy
- CPU that passes the cpu_instrs rom
- GPU
- Keypad
- Timer
- MMU
  - MBC0
  - MBC1
  - MBC3 with realtime-clock
  - MBC5
  - Support for savegames
